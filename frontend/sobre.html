Sobre o Projeto CENA

Objetivo

O CENA é um projeto acadêmico desenvolvido para dar voz à população, permitindo relatar problemas de infraestrutura em suas cidades e sugerir melhorias.

Como Funciona

  • Usuários podem relatar problemas de infraestrutura em suas cidades.
  • As denúncias são exibidas de forma organizada e acessível.

Equipe

O projeto foi desenvolvido por: Nicole

  • Desenvolvimento web (HTML, CSS, JS)
  • Node.js para backend
